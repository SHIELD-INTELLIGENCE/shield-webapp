## Installation & Usage

> ⚠️ Internal use only. External distribution is prohibited.  

1. Clone the repository:  

```bash

git clone https://github.com/SHIELD-INTELLIGENCE/shield-webapp
```

2.Install dependencies:

```bash
# Example
npm install
```

3.Run the software internally:

```bash
npm start
```
